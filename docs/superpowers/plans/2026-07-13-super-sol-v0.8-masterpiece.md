# Super SOL v0.8 Masterpiece Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development
> (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use
> checkbox (`- [ ]`) syntax for tracking.

**Goal:** Produce an immutable `v0.8.0-rc1` Super SOL candidate that actively intervenes only for
GPT-5.6 Sol, remains silent for every other model, and meets stricter latency, privacy, and context
budgets before confirmatory benchmarking.

**Architecture:** Preserve the v0.7 raw-first edit-plus-verification state machine. Add a model gate
at the hook boundary, reduce model-visible context from 220 to 180 code points, and keep safety
guards independent from intervention eligibility. Freeze and publish an RC only after the complete
free release gate passes.

**Tech Stack:** Python 3.9-compatible Codex hooks, Python 3.12 package, pytest, Ruff, basedpyright,
uv, GitHub Actions, GitHub CLI.

## Global Constraints

- Base commit is `98500f5414ba038838b78fc1dc603c36bfe1b39e`.
- Package version is `0.8.0rc1`; plugin version is `0.8.0-rc1`.
- Only normalized model identifier `gpt-5.6-sol` may receive semantic context.
- Terra, Luna, missing model, malformed model, and unknown model identifiers emit zero context.
- Secret and billable-command guards remain active regardless of model eligibility.
- Adaptive prompt submission emits zero context.
- A turn receives at most one residual or repair context after observed edit and verification.
- Every model-visible context is at most 180 Unicode code points.
- Automatic model calls, model switches, subagents, retries, and test reruns are zero.
- No prompt, source, command, output, path, model output, environment value, or secret is persisted.
- No task-pack content may be authored or inspected before the candidate freeze commit.

  Add a helper that runs prompt, successful edit, and first verification with a chosen model:

          "UserPromptSubmit",
          model=model,
          prompt="Fix concurrent refresh cancellation and race conditions",

- [ ] **Step 3: Implement exact model normalization**

  Add this minimal helper near `_diagnostic_control`:

  ```python
  def _model_profile(payload: dict[str, object]) -> str:
      model = payload.get("model")
      if not isinstance(model, str):
          return "observe"
      return "sol" if model.strip().casefold() == "gpt-5.6-sol" else "observe"
  ```

  Store only `model_profile` in request state. Do not persist the raw model string.
