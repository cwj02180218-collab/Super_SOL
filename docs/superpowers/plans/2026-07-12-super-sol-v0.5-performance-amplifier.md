# Super SOL v0.5 Performance Amplifier Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build and verify a `0.5.0rc1` Codex plugin that selectively amplifies GPT-5.6 Terra/medium coding work, plus a four-cell Orca clean-room protocol that compares it with Terra/medium raw, Sol/high raw, and Sol/max raw.

**Architecture:** Super SOL gets a pure deterministic route classifier, four frozen procedure packs, and one evidence-triggered correction inside the existing Codex turn. The public Super_SOL repository owns plugin behavior, packaging, and free Gate 0; the existing `orca-agent-benchmark` repository owns paid-slot scheduling, Orca worktrees, grading, statistics, and the sealed-holdout boundary. Existing v0.3.1 and v0.4 evidence paths remain immutable.

**Tech Stack:** Python 3.12, standard-library hook runtime compatible with `/usr/bin/python3`, Pydantic 2, Typer, pytest, Ruff, basedpyright, uv, stock Codex CLI, Docker, and Orca CLI.

## Global Constraints

- Python package version is `0.5.0rc1`; plugin version is `0.5.0-rc1`.
- Runtime hooks use only the Python standard library and never import the package environment.
- No implementation step starts Codex, an OpenAI API call, a subagent, a grader container, or a paid slot.
- Generic, ambiguous, mixed, explanation-only, and non-action prompts emit no `additionalContext`.
- A specialist route requires weight 2 after Unicode NFKC plus `casefold()` normalization.
- A tie between routes always produces `pass_through`; no precedence breaks a tie.
- Only `terra-v05` has the plugin; every raw cell has zero Super SOL skills.
- Model cells are fixed: Terra/medium raw, Terra/medium v0.5, Sol/high raw, Sol/max raw.
- Gate 1 uses T102, T103, T104, and T112 with four cells and two repetitions: 32 slots.
- Gate 2 uses externally supplied sealed T117-T124 with four cells and two repetitions: 64 slots.
- Completed slots are immutable and are never automatically retried.
- A live run needs the exact standalone user approval plus `--confirm-billable`.
- T109-T116 are disclosed and may not be called an unseen v0.5 holdout.
- Raw prompts, authentication, private graders, session homes, and command outputs never enter public artifacts.

```python
class Route(StrEnum):
    PASS_THROUGH = "pass_through"
    CONCURRENCY_STATE = "concurrency_state"
    SECURITY_BOUNDARY = "security_boundary"
    MIGRATION_COMPATIBILITY = "migration_compatibility"

class RouteDecision:
    route: Route
    score: int

    warning: str | None = None
```

```python
def claim_once(root: Path, name: str) -> bool:
    """Create one owner-only marker and return whether this caller won."""
    marker = root / "claims" / hashlib.sha256(name.encode()).hexdigest()
    marker.parent.mkdir(mode=0o700, parents=True, exist_ok=True)
    try:
        descriptor = os.open(marker, os.O_WRONLY | os.O_CREAT | os.O_EXCL, 0o600)
    except FileExistsError:
        return False
    os.close(descriptor)
    return True
```

```python
class PerformanceCell(StrEnum):
    TERRA_RAW = "terra-raw"
    TERRA_V05 = "terra-v05"
    SOL_HIGH_RAW = "sol-high-raw"
    SOL_MAX_RAW = "sol-max-raw"


@dataclass(frozen=True, slots=True)
class CellSpec:
    model: str
    effort: str
    plugin_enabled: bool


CELL_SPECS = {
    PerformanceCell.TERRA_RAW: CellSpec("gpt-5.6-terra", "medium", False),
    PerformanceCell.TERRA_V05: CellSpec("gpt-5.6-terra", "medium", True),
    PerformanceCell.SOL_HIGH_RAW: CellSpec("gpt-5.6-sol", "high", False),
    PerformanceCell.SOL_MAX_RAW: CellSpec("gpt-5.6-sol", "max", False),
}

_ORDERS = (
    (PerformanceCell.TERRA_RAW, PerformanceCell.TERRA_V05,
     PerformanceCell.SOL_MAX_RAW, PerformanceCell.SOL_HIGH_RAW),
    (PerformanceCell.TERRA_V05, PerformanceCell.SOL_HIGH_RAW,
     PerformanceCell.TERRA_RAW, PerformanceCell.SOL_MAX_RAW),
    (PerformanceCell.SOL_HIGH_RAW, PerformanceCell.SOL_MAX_RAW,
     PerformanceCell.TERRA_V05, PerformanceCell.TERRA_RAW),
    (PerformanceCell.SOL_MAX_RAW, PerformanceCell.TERRA_RAW,
     PerformanceCell.SOL_HIGH_RAW, PerformanceCell.TERRA_V05),
)


def balanced_order(task_index: int, repetition: int) -> tuple[PerformanceCell, ...]:
    order = _ORDERS[task_index % len(_ORDERS)]
    return order if repetition == 1 else tuple(reversed(order))
```
